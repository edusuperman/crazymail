## 🔍 技术发现

### API 特性

1. **JSON-LD 格式**
   - 使用 `@context`, `@id`, `@type` 元数据
   - 符合 Hydra 规范
   - 响应结构清晰

2. **JWT 认证**
   - Token 有效期：1小时
   - 需要在请求头中添加 `Authorization: Bearer {token}`
   - Token 过期后需重新获取

3. **速率限制**
   - 限制：8 QPS（每秒8个请求）
   - 建议延迟：125ms
   - 超限返回 429 状态码

4. **邮箱配额**
   - 每个邮箱：40 MB
   - 无邮件数量限制
   - 永久保留（直到删除）

### 客户端特性

1. **自动化功能**
   - ✅ 自动域名选择
   - ✅ 自动密码生成
   - ✅ Token 自动刷新
   - ✅ 速率限制处理

2. **错误处理**
   - ✅ 指数退避重试
   - ✅ 超时处理
   - ✅ 异常捕获

## 📝 已知问题

### 1. Playwright MCP 自动化

**问题**: Mailfence 登录页面可能返回 404
**影响**: 无法自动化发送邮件
**解决方案**: 
- 使用正确的登录 URL
- 或使用手动发送方式

### 2. 速率限制

**问题**: 8 QPS 限制较严格
**影响**: 快速轮询可能触发限制
**解决方案**: 
- 客户端已添加 125ms 延迟
- 建议轮询间隔 ≥ 10秒
