## ════════════════════════════════════════════
## 核心原则
## ════════════════════════════════════════════

1. **产品优先**：先有能用的产品，再做内容和流量，最后变现
2. **快速迭代**：不追求完美，先上线再优化
3. **数据驱动**：用数据决定下一步，不凭感觉
4. **开源优先**：工具和框架优先选择开源免费方案
5. **人工兜底**：每个自动化模块必须有人工替代方案

## 三阶段开发路线（无时间锁定）
## ════════════════════════════════════════════

### 阶段一：产品 MVP
**目标：上线一个能用的临时邮箱网站**

核心功能：
- 生成临时邮箱地址
- 接收邮件（通过 API 对接 Mail.tm 等）
- 查看邮件内容
- 自动过期清理
- 基础 UI（简洁可用，不追求国风）
- 基础 SEO（SSR、sitemap、meta tags）

验收标准：
- 用户能打开网站 → 生成邮箱 → 收到邮件 → 查看内容
- 网站可被 Google 收录

技术栈：
- 前端：Next.js 15 + Tailwind CSS
- 后端：FastAPI（或 Next.js API Routes 直接处理）
- 邮件 API：Mail.tm / TemporaryMail / Temp-Mail.io
- 部署：Vercel 或 Cloudflare Pages

### 阶段二：流量引擎
**目标：通过 SEO 内容 + 增长黑客获取稳定流量**

核心功能：
- 内容工厂（7阶段 AI 流水线）
- 博客/FAQ 页面
- 关键词自动发现
- 增长黑客（社媒推广，简化版）
- 流量监控（Google Search Console）

验收标准：
- 日均自然搜索流量 > 100 UV
- 内容工厂每周产出 10+ 篇文章

### 阶段三：变现闭环 + 规模化
**目标：实现收入，扩张到 20 站矩阵**

核心功能：
- AdSense 集成
- 联盟营销链接管理
- Premium 功能
- 108好汉系统
- 皇城司隔离
- 国风 Dashboard

验收标准：
- 月收入 > $100
- 3个以上站点同时运营
