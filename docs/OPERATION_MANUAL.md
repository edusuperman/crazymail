# TempMail.Pro 操作手册

## 网站信息
- 网站名称：TempMail.Pro
- 网站地址：http://localhost:3000
- 后端API：http://localhost:8000
- 邮件服务商：Temp-Mail.io
- 最后更新：2026-06-14

## 功能列表

### 1. 邮箱管理

#### 1.1 自动创建邮箱
- **操作**：首次访问网站
- **预期**：系统自动创建随机邮箱地址
- **显示位置**：页面顶部"您的临时邮箱"区域
- **格式**：`随机用户名@随机域名.com`
- **可用域名**：bltiwd.com, wnbaldwy.com, bwmyga.com, ozsaip.com, yzcalo.com, lnovic.com, ruutukf.com, gmeenramy.com

#### 1.2 复制邮箱地址
- **操作**：点击"复制"按钮
- **预期**：邮箱地址复制到剪贴板
- **反馈**：显示"已复制"提示

#### 1.3 随机生成新邮箱
- **操作**：点击"随机"按钮
- **预期**：生成新的随机邮箱地址
- **副作用**：旧邮箱的邮件不再显示

#### 1.4 自定义邮箱
- **操作**：点击"改变"按钮
- **预期**：弹出对话框，可选择域名和输入用户名
- **选项**：
  - 域名下拉菜单：8个可用域名
  - 用户名输入框：可自定义
- **确认**：点击"确认"按钮后创建新邮箱

#### 1.5 删除邮箱
- **操作**：点击"删除"按钮
- **预期**：删除当前邮箱，创建新的随机邮箱

### 2. 邮件管理

#### 2.1 查看邮件列表
- **位置**：页面下方"消息"标签页
- **显示**：发件人、主题、时间
- **排序**：按时间倒序（最新在前）
- **刷新**：每5秒自动刷新

#### 2.2 打开邮件详情
- **操作**：点击邮件列表中的邮件
- **预期**：弹出邮件详情对话框
- **显示**：主题、发件人、时间、正文
- **格式**：支持HTML和纯文本

#### 2.3 删除邮件
- **操作**：在邮件详情中点击"删除邮件"按钮
- **预期**：
  - 如果成功：邮件从列表消失，显示"已删除"提示
  - 如果失败：显示"删除失败：该服务不支持删除邮件，请等待邮件自动过期"
- **注意**：Temp-Mail.io不支持删除邮件，会显示失败提示

#### 2.4 收藏邮件
- **操作**：在邮件详情中点击"收藏"按钮
- **预期**：邮件添加到"已保存"标签页

#### 2.5 刷新邮件
- **操作**：点击"刷新"按钮
- **预期**：立即刷新邮件列表

### 3. 界面功能

#### 3.1 语言切换
- **操作**：点击右上角语言按钮
- **选项**：中文、英文、法语、德语、日语、韩语
- **预期**：界面语言切换

#### 3.2 响应式布局
- **测试**：调整浏览器窗口大小
- **预期**：界面自适应，功能正常

## 操作流程

### 流程1：接收验证邮件
1. 打开网站，记录邮箱地址
2. 在需要验证的网站输入邮箱地址
3. 返回TempMail.Pro，等待邮件到达（最多30秒）
4. 点击邮件查看验证码
5. 复制验证码完成验证

### 流程2：测试邮件收发
1. 打开网站，记录邮箱地址
2. 从其他邮箱发送测试邮件
3. 等待邮件显示（5-10秒）
4. 点击邮件查看内容
5. 验证邮件格式和内容正确

### 流程3：自定义邮箱
1. 点击"改变"按钮
2. 选择域名（如@ozsaip.com）
3. 输入自定义用户名（如testuser）
4. 点击"确认"
5. 验证新邮箱地址为testuser@ozsaip.com

## 已知限制

1. **不支持删除邮件**：Temp-Mail.io API不支持删除，点击删除会显示错误提示
2. **邮箱过期**：临时邮箱会在一段时间后过期
3. **不支持发送邮件**：只能接收，不能发送
4. **不支持附件下载**：只能查看附件信息，不能下载

## 常见问题

### Q1：为什么收不到邮件？
A1：
- 检查邮箱地址是否正确
- 等待5-10秒，邮件可能有延迟
- 点击"刷新"按钮手动刷新
- 某些网站可能屏蔽临时邮箱域名

### Q2：为什么删除邮件失败？
A2：Temp-Mail.io不支持删除邮件功能，请等待邮件自动过期。

### Q3：邮箱地址会过期吗？
A3：是的，临时邮箱会在一段时间后过期。如需长期使用，请保存重要邮件内容。

### Q4：可以自定义邮箱域名吗？
A4：可以。点击"改变"按钮，选择域名下拉菜单中的可用域名。

### Q5：支持哪些邮件格式？
A5：支持HTML和纯文本格式。HTML邮件会正确渲染样式。

### Q6：iOS/Android应用什么时候上线？
A6：这些功能已规划，正在开发中。页面已标注"Coming Soon"。
