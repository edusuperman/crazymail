# ADR-001: 项目战略调整 — 重新对齐终极目标

**日期：** 2026-06-11
**状态：** 已接受
**决策者：** 用户

## 背景

原方案（Claude 编写）将"内容工厂（7阶段流水线）"作为开发重点，但项目的终极目标是"靠临时邮箱网站矩阵赚钱"。

原方案存在目标偏差：
- 跳过了产品（临时邮箱网站）直接做内容生产
- 缺少变现手段（AdSense、联盟营销）
- 缺少流量获取机制

## 决策

将开发路线调整为三阶段：

1. **阶段一：产品 MVP** — 先有能用的临时邮箱网站
2. **阶段二：流量引擎** — SEO 内容 + 增长黑客
3. **阶段三：变现闭环** — AdSense + 联盟 + 规模化

原方案的内容工厂、108好汉、皇城司、国风 Dashboard 全部推迟到阶段二/三。

## 影响

- 开发优先级重新排序
- 阶段一聚焦邮件 API 对接和前端 UI
- SEO/GEO 从第一天开始考虑
