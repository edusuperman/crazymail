## 建议和改进

### 对用户的建议

1. **优先使用 TemporaryMail**: 接收速度最快
2. **及时读取邮件**: 邮件可能会被清理
3. **使用 getEmail 获取完整信息**: checkInbox 只返回基本信息
4. **合理设置轮询间隔**: 建议 5秒

### 对客户端的改进

1. **自动获取完整主题**: 在 `check_inbox` 中自动调用 `get_email_detail`
2. **添加邮件缓存**: 避免重复请求
