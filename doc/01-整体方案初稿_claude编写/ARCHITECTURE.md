**质检节点（岳飞9重）**：
1. AI味检测（Originality.ai API，目标 < 30分）
2. SEO评分（Surfer SEO API，目标 > 75分）
3. E-E-A-T信号（作者署名/数据引用/经验描述）
4. 字数/结构（目标 1500-3000词，H2/H3规范）
5. 内链建议（站内相关文章链接）
6. 图片alt文本
7. Schema markup 完整性
8. 去重检测（与站内已发内容）
9. 人味评分（口头禅/个人观点/轻微错别字存在性）

## 3. 人格系统架构

### 108好汉（公开社媒面向，人味 85%）

```
persona_profile/
├── soul.json           # SoulSpec 主清单
├── SOUL.md             # 核心灵魂：价值观/quirks/信念
├── IDENTITY.md         # 身份：职业/城市/背景
├── STYLE.md            # 写作风格：口头禅/句式/emoji偏好
├── MEMORY.md           # 成长日志（自动追加）
├── HEARTBEAT.md        # 行为节律：时区/活跃时间/频率
├── HISTORY.md          # 历史发帖记录
├── GROWTH.md           # 成长等级 + 里程碑
└── examples/
    ├── good_posts.md   # 该人格的典型发帖
    └── bad_posts.md    # 该人格绝不会发的内容
```

**成长等级**：
- Lv.1 刚上梁山（0-30天）：低频互动，建立账号可信度
- Lv.2 八十万禁军教头（30-90天）：发原创内容，固化垂类标签
- Lv.3 梁山好汉（90-180天）：有粉丝基础，开始有机推广
- Lv.4 封神入册（180天+）：高权威账号，外链价值最高

### 编制内官员（内部流水线，人味 35%）

专业人格，核心是领域知识积累：SOUL.md + EXPERTISE.md + FEEDBACK.md

## 4. 皇城司：安全隔离架构

**7层隔离矩阵**：

| Layer | 隔离维度 | 关键规则 |
|-------|---------|---------|
| 1 | 物理层 | 20站点分散至5家主机商，不共享 /24 子网 |
| 2 | 网络层 | 住宅代理池，IP与好汉归属地地理匹配 |
| 3 | 身份层 | 每5站点独立注册商/支付/WHOIS |
| 4 | 浏览器层 | Dolphin Anty 独立指纹Profile |
| 5 | 行为层 | 行为节律引擎，按时区/频率差异化 |
| 6 | 内容层 | 零内链 + 独立写作风格 + 不共享CDN路径 |
| 7 | 资金层 | 按Niche分组 AdSense，不归集单账户 |

## 5. 十二道金牌：熔断系统

分级自动触发 → 最终人工接管协议

每道金牌有：触发条件 + 执行动作 + 通知方式
第一道（轻度）→ 第十二道（全停，纯人工接管）

详见：modules/GOLD_MEDAL.md

## 6. Dashboard 可视化架构

**主画布**：横向滚动的宋代城镇（清明上河图）
- PixiJS 8 主渲染引擎
- WebSocket 实时接收后端事件驱动动效
- 飞鸽：PixiJS Particle System（任务流转动效）
- 官员动作：Sprite Animation（简笔画风格）
- 皇城司彩蛋：画面角落暗门，点击进入安全控制台

**数据面板**：React Flow + Recharts
- 7阶段流水线实时状态
- 20站点流量/收益趋势
- 108好汉账号健康状态地图

## 7. 数据流

```
事件产生（Agent完成任务）
    ↓
FastAPI → Redis Pub/Sub 发布事件
    ↓
WebSocket Server 推送到前端
    ↓
PixiJS 场景响应（飞鸽起飞动效）
    ↓
React Flow 流程图节点状态更新
    ↓
数据库记录（PostgreSQL）
```
