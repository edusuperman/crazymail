# CrazyMail 人物矩阵详细设计

## 1. 整体逻辑
- 编制内（朝廷）：核心内容生产、7阶段流水线
- 编外（108好汉）：增长黑客推广可视化

## 2. 编制内人物矩阵 (北宋同期名人)

**核心角色分配** (一对多关系为主)：

- **宋徽宗赵佶**：总管 + 最终决策 (人工把关象征)
- **李纲**：研究 + 关键词阶段
- **张择端**：大纲 + SERP分析 (清明上河图隐喻)
- **周邦彦 / 李清照**：内容写作 (可并行)
- **岳飞**：9重质检 (忠诚把关)
- **韩世忠**：链接建设 + 部分增长
- **种师道**：SEO/GEO战略优化

**关系**：混合使用，一人可负责多个站点/阶段

## 3. 编外108好汉矩阵
- 仅增长黑客模块
- 分组展示：按地理/平台 (一对多：一个好汉代表多个虚拟身份)

## 4. 隔离策略
- 编制内人物：站点间混合但内容严格分离
- 编外好汉：推广身份严格按4-5集群隔离
