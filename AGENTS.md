## 终极目标（永不改变）

## ════════════════════════════════════════════
## 三阶段路线（在 STRATEGY.md 中定义当前阶段）
## ════════════════════════════════════════════

### 阶段一：产品 MVP
- 核心：临时邮箱的生成、接收、查看、过期
- 技术：Next.js 15 + FastAPI + 邮件 API
- 目标：上线一个能用的产品

### 阶段二：流量引擎
- 核心：内容工厂（7阶段流水线）+ SEO + 增长黑客
- 技术：CrewAI + LangGraph + 108好汉系统
- 目标：日均自然流量 > 100 UV

### 阶段三：变现闭环 + 规模化
- 核心：AdSense + 联盟营销 + Premium + 多站矩阵
- 技术：皇城司隔离 + 国风 Dashboard
- 目标：月收入 > $3000

## ════════════════════════════════════════════
## 技术栈（全阶段通用）
## ════════════════════════════════════════════

**前端：** Next.js 15 (App Router) + TypeScript + Tailwind CSS
**后端：** Python 3.12 + FastAPI + uv
**数据库：** Supabase（PostgreSQL）
**LLM：** MiMo v2.5 Pro（主力）+ Agnes AI（免费批量）
**部署：** Vercel 或 Cloudflare Pages

**阶段二额外：** CrewAI + LangGraph + Redis
**阶段三额外：** Dolphin Anty + 代理池

## ════════════════════════════════════════════
## 代码规范（全阶段通用）
## ════════════════════════════════════════════

**Python：** 3.12+，类型注解完整，uv 管理依赖，snake_case，中文注释
**TypeScript：** 严格模式，禁止 any，kebab-case 文件名，PascalCase 组件名
**通用：** 配置走环境变量，禁止硬编码密钥

## ════════════════════════════════════════════
## SEO/GEO 规范（全阶段通用）
## ════════════════════════════════════════════

每个页面必须：
- SSR（服务端渲染）
- Meta tags + OG tags
- JSON-LD 结构化数据
- 语义化 HTML
- 干净的 URL
- 图片 alt 文本
- E-E-A-T 信号

详见 `SEO_GEO_GUIDE.md`
